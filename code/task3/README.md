# Task 3a Streamlit app

## Setup

```bash
cd /path/to/project
/path/to/venv/bin/pip install -r code/task3/requirements.txt
```

## Run

```bash
cd /path/to/project
/path/to/venv/bin/streamlit run code/task3/app_3a.py
```

Default CSV path: `data/output/routes.csv`. Override it in the sidebar if needed.
